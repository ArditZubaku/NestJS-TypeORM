### A small example app using [Nest](https://github.com/nestjs/nest) framework with [TypeORM](https://typeorm.io/) and [Jest](https://jestjs.io/)

## Installation

```bash
yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# e2e tests watch mode
$ yarn run test:e2e:watch

# test coverage
$ yarn run test:cov
```
